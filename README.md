# QMK Userspace

This is the QMK Userspace for the Bastard Keyboards keymaps. Customsed by Noah for the Charybdis 4x6. Layout features QMK Auto mouse, per layer RGB and the VIA macro fix. 

You can read how to compile your own keymap on the official docs here: [https://docs.bastardkb.com/fw/compile-firmware.html](https://docs.bastardkb.com/fw/compile-firmware.html).
